User Behaviour Analysis Overview

Artifacts ▼

UserAssist Applications Opened
NTUSER.DAT\...\Explorer\UserAssist\{GUID}
  • Tracks applications launched through Windows Explorer (GUI launches only — not command line).
  • Data is encoded in ROT-13 — Registry Explorer decodes this automatically.
  • Records execution count, last execution time, and focus time for each application.
  • Useful for proving a user knowingly ran a specific application during the attack timeframe.
RecentDocs Files & Folders Opened
NTUSER.DAT\...\Explorer\RecentDocs
  • Tracks recently accessed files and folders opened via Windows Explorer.
  • Organised by file extension subkeys — each extension has its own MRU list.
  • Records the file name and the order in which files were accessed.
  • Can reveal access to sensitive documents, executables, or files on removable media.
Shellbags Locations Browsed
USRCLASS.DAT\...\Shell\Bags
  • Records folder browsing history including window size, position, and view settings for each folder visited.
  • Persists even after folders or files are deleted — can prove access to locations that no longer exist.
  • Can reveal access to network shares, removable drives, and external storage.
  • Best parsed with ShellBags Explorer or Registry Explorer.
Open/Save MRU Files Opened via Dialog
NTUSER.DAT\...\ComDlg32\OpenSavePIDlMRU
  • Tracks files opened or saved through the Windows Open/Save dialog box.
  • Organised by file extension — each has its own MRU list with full file paths.
  • Records the full path of the file, which can reveal directory structures and drive letters.
  • Useful for identifying access to files on USB drives or network paths.
Last-Visited MRU Applications Used to Open Files
NTUSER.DAT\...\ComDlg32\LastVisitedPIDlMRU
  • Tracks which application was used to open or save a file and the last directory it browsed to.
  • Complements Open/Save MRU — together they can reconstruct the full file access story.
  • Can link a specific tool (e.g. a text editor or archiver) to a suspicious file location.
