☐ Review Alias & Users bookmarks — export CSV to Timeline Explorer

UserAssist ▼
☐ Identify all user NTUSER.DAT hives to analyse
☐ Review GUID {CEBFF5CD} — applications, files & objects accessed
☐ Review GUID {F4E57C4B} — shortcut links used to start programs
☐ Run userassist RegRipper plugin per user hive
RecentDocs ▼
☐ Review root key — most recently accessed files across all types
☐ Check extension subkeys for suspicious file types (.exe, .ps1, .zip)
☐ Check Folder subkey for recently visited directories
☐ Run recentdocs RegRipper plugin per user hive
ShellBags ▼
☐ Run shellbags plugin against USRCLASS.DAT (primary)
☐ Run shellbags plugin against NTUSER.DAT (secondary)
☐ Load both hives in ShellBag Explorer and export CSV to Timeline Explorer
☐ Sort by Last Write Time and review USB, network & deleted folder paths
MFT ▼
☐ Parse $MFT with MFTECmd and export to CSV
☐ Open CSV in Timeline Explorer and filter by attack timeframe
☐ Filter InUse = False in Timeline Explorer to identify deleted file entries
☐ Filter SI<FN = True in Timeline Explorer to flag timestomping
☐ Check timestamps rounded to exact seconds — indicator of manual timestomping
USN Journal ▼
☐ Parse $J stream with MFTECmd and export to CSV
☐ Filter FILE_DELETE entries in Timeline Explorer during attack window
☐ Look for Rename → Delete pattern indicating Recycle Bin then permanent deletion
☐ Correlate USN entries with $MFT file reference numbers
BAM ▼
☐ Run BAM RegRipper plugin against SYSTEM hive
☐ Review per-user SID entries for executed programs
☐ Check for LOLBins and tools run from suspicious locations
☐ Cross-reference execution timestamps with attack timeframe
ShimCache ▼
☐ Parse ShimCache with AppCompatCacheParser and export to CSV
☐ Review entries for executables in suspicious locations
☐ Note: Win10 entries do not confirm execution — corroborate with BAM / Amcache
☐ Check for deleted tools still present in cache
Amcache ▼
☐ Parse Amcache.hve with AmcacheParser and export to CSV
☐ Review UnassociatedFileEntries for standalone executed files
☐ Submit SHA1 hashes of suspicious files to VirusTotal
☐ Check first execution timestamps against attack timeframe
Prefetch ▼
☐ Confirm Prefetch is enabled on the system
☐ Parse all .pf files with PECmd and export to CSV
☐ Review LastRun timestamps and execution counts during attack window
☐ Check for same executable run from multiple paths (different hashes)
☐ Check volume serial number — execution from USB or removable media
Run Keys ▼
☐ Review HKLM Run & RunOnce keys (SOFTWARE hive)
☐ Review HKCU Run & RunOnce keys for each user (NTUSER.DAT)
☐ Check WOW6432Node Run key for 32-bit persistence entries
☐ Flag entries pointing to Temp, AppData, Downloads or using LOLBins
☐ Check last write time of Run keys to place persistence in timeline
Startup Folders ▼
☐ Check per-user startup folder for each user profile
☐ Check all-users startup folder (ProgramData)
☐ Inspect any .lnk shortcut files for their target path
☐ Verify Shell Folders registry keys point to expected paths
☐ Check $MFT timestamps of startup folder files against attack timeframe
Windows Services ▼
☐ Run services RegRipper plugin against SYSTEM hive
☐ Review services with Start=2 (Automatic) created during attack window
☐ Check ImagePath for non-standard locations or LOLBin usage
☐ Flag services running as LocalSystem with unknown or suspicious names
☐ Check last write time of service registry keys against attack timeframe
Scheduled Tasks ▼
☐ Run taskcache RegRipper plugin against SOFTWARE hive
☐ Browse C:\Windows\System32\Tasks\ for suspicious XML files
☐ Check $MFT created timestamps of XML files against attack window
☐ Inspect <Actions> elements for LOLBin usage or suspicious paths
☐ Check TaskCache for entries with no matching XML file (deleted tasks)
Windows Defender Events ▼
☐ Parse Defender event log with EvtxECmd and export to CSV
☐ Filter EventID 1116 — malware detections (threat name, path, timestamp)
☐ Filter EventID 5000 / 5001 — protection enabled/disabled (look for disable → re-enable pattern)
☐ Filter EventID 5007 — configuration changed (check old vs new values)
☐ Check if disable events preceded malware activity in the timeline
Service Installs ▼
☐ Parse System.evtx with EvtxECmd and export to CSV
☐ Filter EventID 7045 — review all new service installs
☐ Check Service File Name for suspicious paths or LOLBin usage
☐ Cross-reference 7045 timestamps with attack timeline
Authentication Events ▼
☐ Parse Security.evtx with EvtxECmd and export to CSV
☐ Filter 4624 — successful logins (focus on Type 10 RDP and Type 3 Network)
☐ Filter 4625 — failed logins (brute-force pattern: many failures then success)
☐ Filter 4720 / 4732 — new account created or added to Administrators
☐ Filter 4648 — explicit credential use (RunAs / lateral movement)
PowerShell ▼
☐ Open Microsoft-Windows-PowerShell%4Operational.evtx in Event Log Explorer
☐ Filter 4104 — script block logging; narrow to Level = Warning for auto-flagged suspicious blocks
☐ Reassemble multi-part scripts via shared ScriptBlockId (MessageNumber of MessageTotal)
☐ Filter 4103 — module/pipeline logging (commands and bound parameter values)
☐ Open Windows PowerShell.evtx and filter 400 — check HostApplication and EngineVersion
☐ Cross-reference PowerShell activity with the attack timeframe
Sysmon ▼
☐ Open Microsoft-Windows-Sysmon%4Operational.evtx in Event Log Explorer
☐ Filter EventID 1 — process creation (command line, parent process, hashes)
☐ Grab ProcessGuid from a suspicious process and pivot to its 3 / 11 / 22 events
☐ Filter EventID 3 — network connections (suspicious destination IPs / beaconing)
☐ Filter EventID 11 — file creates (dropped payloads in Temp / AppData / ProgramData)
☐ Filter EventID 12 / 13 — registry key/value changes (Run keys and other persistence)
☐ Filter EventID 22 — DNS queries (C2 domains / DGA)
