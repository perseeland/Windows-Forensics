Windows Forensics Notes

Data Collection ▼

1 Mount with Arsenal Image Mounter
  • Open Arsenal Image Mounter and load the .vhd disk image.
  • In the mount dialog, set the following options:
    • Select Disk device, write temporary
    • Check "Store differencing data in host RAM only (not in a file)"
  • Click OK to mount.
  • Check This PC or Disk Management for the assigned drive letter.
Using Write Temporary + RAM keeps the original image forensically clean — no writes touch the source.
2 Triage Collection with KAPE
  • Launch gkape.exe (the GUI version of KAPE).
  • Click on Target options.
  • Set Target source to the mounted image drive from Arsenal Image Mounter.
  • Set Target destination to C:\Cases.
  • Select the KapeTriage package.
