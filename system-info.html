System Information

Path Variables

Computer Name ▼

HKLM\System\CurrentControlSet\Control\Computername\
  • Look for the ComputerName value to identify the hostname of the system.
  • Useful for confirming machine identity and correlating with network logs or domain records.
RegRipper

Windows Version ▼

HKLM\Software\Microsoft\Windows NT\CurrentVersion\
  • Ensure you navigate to Windows NT and not the non-NT Windows key.
  • Key values: ProductName, CurrentBuild, ReleaseId, and InstallDate.
  • InstallDate is stored as a Unix timestamp — note it as potential evidence of system setup time.
RegRipper

Timezone ▼

HKLM\System\CurrentControlSet\Control\TimeZoneInformation\
  • Look for TimeZoneKeyName to identify the configured timezone.
  • Critical for accurately interpreting all timestamps — event log and filesystem timestamps must be adjusted relative to this value.
RegRipper

Network Information ▼

HKLM\System\CurrentControlSet\Services\Tcpip\Parameters\Interfaces\{interface-name}
  • Each subkey under Interfaces represents a network adapter identified by a GUID.
  • Look for IPAddress, DhcpIPAddress, DhcpServer, and DhcpSubnetMask.
  • Can reveal previously assigned IP addresses even if the configuration has since changed.

Shutdown Time ▼

HKLM\System\ControlSet001\Control\Windows\ShutdownTime
  • Stores the last clean shutdown time as a FILETIME binary value.
  • Helps establish a timeline — determines when the suspect last powered off the machine.
  • Only updated on a clean shutdown, not on crash or power loss.
RegRipper

Defender Settings ▼

HKLM\Software\Microsoft\Windows Defender\
  • Check DisableAntiSpyware and DisableAntiVirus — a value of 1 indicates Defender was disabled.
  • Note the last write time of the key — if the value is 0 (enabled) but the last write time is recent or suspicious, it suggests Defender was re-enabled after previously being turned off, potentially to cover tracks.
  • Tampered Defender settings can indicate deliberate evasion by a threat actor.
  • Check Exclusions subkeys for any paths or processes whitelisted to bypass scanning.
