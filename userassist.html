UserAssist Analysis

Overview ▼

  • UserAssist tracks applications, files, and links accessed by the user through Windows Explorer.
  • Analysis must be performed per user — each user account has its own NTUSER.DAT hive containing their UserAssist data.
  • All entries are encoded in ROT-13 — Registry Explorer and the RegRipper userassist plugin decode this automatically.
  • Each entry records run count, last run time, focus count, and focus time — providing a detailed picture of application usage.
  • Located at: NTUSER.DAT\Software\Microsoft\Windows\CurrentVersion\Explorer\UserAssist\{GUID}\Count

GUIDs ▼

{CEBFF5CD-ACE2-4F4F-9178-9926F41749EA}
  • Contains a list of applications, files, links, and other objects that have been accessed by the user.
  • The primary GUID for application execution evidence — covers programs launched directly and files opened.
  • Each entry includes the full path of the executable or file (ROT-13 encoded).
{F4E57C4B-2036-45F0-A9AB-443BCFE33D9F}
  • Lists the shortcut links (.lnk files) used to start programs.
  • Captures application launches made via desktop shortcuts, taskbar pins, and Start Menu entries.
  • Complements the first GUID — together they provide a complete view of how programs were launched.
Both GUIDs should be reviewed together for a complete picture of user activity. One captures direct execution, the other captures shortcut-based launches.

RegRipper ▼

  • The userassist plugin parses both GUIDs and decodes all ROT-13 entries automatically.
  • Run against each user's NTUSER.DAT hive — repeat for every user account found on the system.
  • Output includes the decoded program name, run count, and last execution timestamp.
RegRipper
Run against a user's NTUSER.DAT:
rip.exe -r C:\Cases\Analysis\Registry\NTUSER.DAT -p userassist > C:\Cases\Analysis\Registry\RR_Output\userassist.txt && type C:\Cases\Analysis\Registry\RR_Output\userassist.txt
